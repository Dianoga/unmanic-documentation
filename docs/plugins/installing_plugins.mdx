Plugins are hosted on repositories.

Unmanic comes configured to be able to download Plugins from the Official repository.

To install a Plugin, follow these steps:

1. On the Plugins tab, click the **ADD NEW** button.

![Plugins - open add new button](/img/unmanic-plugins-add-new-button.png "unmanic-plugins-add-new-button")

2. Click the **Check for updates** button.

![Plugins - install screen](/img/unmanic-plugins-install-screen.png "unmanic-plugins-install-screen")

3. Install your desired Plugin via the **Install** button.

4. Close this window and return to the Plugins tab.

![Plugins - enable](/img/unmanic-plugins-enable.png "unmanic-plugins-enable")

5. Enable the Plugin
