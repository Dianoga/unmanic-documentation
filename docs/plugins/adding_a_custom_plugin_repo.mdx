## Adding a repo

To add a custom repository located on GitHub, follow these steps:

1. On the Plugins tab, click the **ADD NEW** button.

![Plugins - open add new button](/img/unmanic-plugins-add-new-button.png "unmanic-plugins-add-new-button")

2. Click the **ADD REPOSITORY** button.

3. Add the maintainers URL into the *New Repository* text input.

![Plugins - adding a custom repo](/img/unmanic-plugins-add-custom-repositories.png "unmanic-plugins-add-custom-repositories")

4. Click the **SAVE** button.

This will automatically refresh the available plugins to include the ones in the added repository.

---

## Removing a repo

You can remove a custom repo by clicking on the **REPOSITORIES LIST** button and then selecting **REMOVE**.

![Plugins - display custom repos](/img/unmanic-plugins-add-custom-repositories-list.png "unmanic-plugins-add-custom-repositories-list")
