Plugins are hosted on repositories.

Unmanic comes configured to be able to download plugins from the [official repository](https://github.com/Unmanic/unmanic-plugins/tree/official).

To install a plugin, follow these steps:

1. On the **Plugins** tab, click the **ADD NEW** button.

<img className={"screenshot"} src={require('/img/unmanic-plugins-add-new-button.png').default} />

2. Click the **REFRESH REPOSITORIES** button.

<img className={"screenshot"} src={require('/img/unmanic-plugins-install-refresh-repos.png').default} />

3. Install your desired plugin via the **Install** button.

<img className={"screenshot"} src={require('/img/unmanic-plugins-install-download.png').default} />

To enable a plugin for a library, see [Configuring Libraries](/docs/configuration/libraries/configure_libraries#plugins)
